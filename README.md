# rpi-cd-ripper

Automatic CD Ripping solution, inspired by https://github.com/aac/raspberrypi-cd-ripper

```shell
sudo apt -y install wget lame flac abced python3 python3-requests
sudo wget https://raw.githubusercontent.com/9001/copyparty/refs/heads/hovudstraum/bin/u2c.py -O /usr/local/bin/u2c.py
sudo chmod +x /usr/local/bin/u2c.py
```
